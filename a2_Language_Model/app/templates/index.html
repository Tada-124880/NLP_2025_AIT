How it works:

This web application uses a pre-trained LSTM-based language model to generate text. When you enter a prompt (e.g., "Harry Potter is"), the model predicts and generates the continuation of the text.

After you submit your input, the Flask backend processes it and sends back the generated continuation, which will be displayed here.
